# spectrwm.conf
my dotfiles for the spectrwm window manager
feel free to fork but I will not accept changes                            
#dependencies                                      
i3lock-fancy                                       
pcmanfm                            
st                          
dmenu                        
spectrwm                       
mononoki nerd font
conky                                                          

#installing                       
git clone https://github.com/nathaniel-olmsted/spectrwm.conf                    
put .spectrwm.conf in your user's home folder                      
done!
